# Affiliate Site Workflow

When a new task is received:

1. Read task file.
2. Perform product research.
3. Generate article draft.
4. Generate SEO metadata.
5. Generate FAQ section.
6. Validate internal links.
7. Save article in articles/drafts.
8. Do not modify published articles unless explicitly instructed.

Rules:

* Use beginner-friendly language.
* Prioritize buyer intent keywords.
* Include comparison tables.
* Include pros and cons.
* Include final recommendation.
