# Curated Picks

A small static affiliate website published with GitHub Pages.

## Main Files

- `index.html`: homepage
- `products/products.json`: product catalog
- `best-wireless-earbuds-Under-2000.html`: current buying guide
- `assets/css/style.css`: site styles
- `assets/js/app.js`: homepage rendering and filters
- `scripts/validate-site.mjs`: automatic safety checks

## Make a Safe Change

1. Edit the required file.
2. Run `npm test`.
3. Preview the changed page in a browser.
4. Review the Git diff.
5. Commit and push only after the checks pass.

## Run Checks

Node.js 20 or newer is recommended.

```bash
npm test
```

No third-party packages are required.
